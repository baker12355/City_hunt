# Searching_Game

> 這是一個基於物件辨識的遊戲，啟發的遊戲為"支援前線"。玩家分成紅藍兩隊，系統會出題目給兩隊玩家，隊員找到系統指定的物件並放到鏡頭前，經系統辨認正確即可得分，時間結束前得分最高的隊伍勝利。背後使用的模型為ssd_mobilenet(RCNN的輕量版)、以及OpenCV的影像串流、介面使用Tkinter。
